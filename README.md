# Del SLA a la IA

Research OS para construir un framework de ingenieria de servicios basado en datos.

## Inicio rapido en VS Code

1. Clona el repositorio.
2. Abre la carpeta en VS Code.
3. Lee primero `PROJECT_MEMORY.md`.
4. Luego lee `AGENTS.md`, `ROADMAP.md` y `CURRENT_TASK.md`.
5. Pide a Codex o ChatGPT: continua el Research OS siguiendo las reglas del repositorio.

## Framework inicial

DDIA:
- Define
- Design
- Instrument
- Analyze

## Serie editorial

Del SLA a la IA conecta mesa de ayuda, infraestructura, KPI, OKR, observabilidad e inteligencia artificial aplicada a servicios tecnologicos.
