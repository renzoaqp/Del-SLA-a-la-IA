# Como continuar en VS Code

## Paso 1
Clonar el repositorio:

git clone https://github.com/renzoaqp/Del-SLA-a-la-IA.git

## Paso 2
Abrir la carpeta en VS Code.

## Paso 3
Pedir a Codex o ChatGPT:

Lee PROJECT_MEMORY.md, AGENTS.md, ROADMAP.md y CURRENT_TASK.md. Luego revisa la estructura del repositorio y continua el desarrollo del Research OS Del SLA a la IA.

## Paso 4
Trabajar siempre registrando cambios importantes en DECISIONS.md.
