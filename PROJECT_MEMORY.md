# PROJECT MEMORY

Proyecto: Del SLA a la IA.

Objetivo: construir un Research OS para desarrollar un framework de ingenieria de servicios basado en datos.

Framework base: DDIA.

DDIA significa:
- Define
- Design
- Instrument
- Analyze

Linea editorial:
- Articulo 1: servicios y mesa de ayuda.
- Articulo 2: infraestructura, redes y plataformas.
- Articulo 3: KPI y OKR en operaciones reales.
- Articulo 4: observabilidad, datos e inteligencia artificial aplicada.

Principios:
- No inventar referencias.
- Usar fuentes academicas y tecnicas verificables.
- Diferenciar evidencia, interpretacion y opinion.
- Mantener trazabilidad entre conceptos, datos y decisiones.
- Escribir para LinkedIn con rigor profesional.

Uso en VS Code:
1. Abrir el repositorio.
2. Leer este archivo primero.
3. Leer AGENTS.md para reglas de trabajo.
4. Revisar CURRENT_TASK.md para saber que hacer.
5. Registrar decisiones importantes en DECISIONS.md.
