# SLO - Service Level Objective

Un SLO es un objetivo interno de nivel de servicio.

## Relacion con SLA
El SLO normalmente debe ser mas exigente que el SLA para dar margen operativo.

Ejemplo:
- SLO interno: 99.95 por ciento de disponibilidad.
- SLA externo: 99.9 por ciento de disponibilidad.

## Uso
Sirve para gestionar el servicio antes de incumplir compromisos externos.

## Pregunta clave
Que objetivo interno permite operar con margen razonable?
