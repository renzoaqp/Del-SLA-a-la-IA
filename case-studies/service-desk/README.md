# Caso de estudio: Mesa de ayuda

## Objetivo
Aplicar DDIA a una mesa de ayuda real o simulada.

## Preguntas
- Que servicio se entrega?
- Que tickets se atienden?
- Cuales son las prioridades reales?
- Que datos existen?
- Que SLA se prometio?
- Que SLO interno permitiria cumplirlo?

## Entregables
- Catalogo de SLI.
- Propuesta de SLO.
- SLA defendible.
- KPIs derivados.
- OKRs sugeridos.
