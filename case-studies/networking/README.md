# Caso de estudio: Networking

## Objetivo
Aplicar DDIA a un servicio de conectividad o red corporativa.

## Preguntas
- Que servicio compra realmente el negocio?
- Desde donde se mide?
- Que ventanas de mantenimiento existen?
- Que eventos se excluyen?
- Cual es el costo de mejorar disponibilidad?

## Entregables
- Catalogo de SLI.
- SLO interno.
- SLA externo.
- Dashboard propuesto.
- Decisiones de inversion.
